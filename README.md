# Shopping_Online
Web online shopping
# Cài đặt:
  - NET 8.0
  - Các Package cần thiết trong file Shopping_Online.csproj
  - Kết nối Database bằng "ConnectionStrings" trong file appsettings.json
# Chạy:
  - dotnet watch run
# Lưu ý:
  - Bạn phải tạo tài khoản Role 'Admin' từ trong CSDL trước khi khởi chạy
